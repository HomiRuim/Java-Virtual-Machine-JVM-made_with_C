Executa o desassembleamento do Bytecode contido em um CodeAttribute.

Le o array de bytes do codigo, traduz cada opcode para o mnemônico correspondente, le e resolve os argumentos, e popula a estrutura DisasmOutput.

Parameters
cfEstrutura ClassFile completa (necessaria para resolucao do CP).
code_attrO atributo Code do metodo (contem o array de bytes).
outputPonteiro para a estrutura onde os resultados serao armazenados.
Returns
true em sucesso, false em erro de alocacao de memoria.
