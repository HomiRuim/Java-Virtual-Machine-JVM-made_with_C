typedef int(* OpcodeHandler) (Frame *frame, const CliOptions *options)

Assinatura de um manipulador de opcode.

Cada função recebe o Frame atual e as opções de CLI (para modo debug). É responsável por:

  1. Ler os operandos do bytecode (se houver) a partir de frame->pc.
  2. Executar a operação (manipular a pilha de operandos e/ou variáveis locais).
  3. Atualizar o Program Counter (frame->pc) para o próximo opcode.
Parameters
frameO Frame de execução atual.
optionsAs opções de CLI (para debug).
Returns
0 em caso de sucesso, 1 para sinalizar return, ou um código negativo para erro.

Executa o método principal (main) da classe carregada.

Esta função implementa o interpretador principal da JVM:

  1. Busca o método main
  2. Obtém o Code Attribute
  3. Cria o Frame de execução
  4. Loop de execução usando a Dispatch Table
  5. Limpeza de memória
